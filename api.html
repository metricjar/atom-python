put_event(stream, data, method='POST', auth_key='')[source]¶

Send a single event to Atom API

This method exposes two ways of sending your events. Either by HTTP(s) POST or GET.

Parameters:
  • method (str) – the HTTP(s) method to use when sending data - default is POST
  • stream (str) – Atom Stream name
  • data (str) – a string of data (payload) to send to the server
  • auth_key (str) – hmac auth key
Returns:

requests response object
